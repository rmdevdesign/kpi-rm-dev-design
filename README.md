# KPI RM Dev Design

Outil local pour suivre les indicateurs utiles à l'activité `rmdev.design`.

## Lancer

Ouvre `index.html` dans ton navigateur.

Aucun serveur n'est requis. Les données sont conservées localement dans le navigateur via `localStorage`.

## KPI suivis

- Argent gagné
- Missions actives
- Devis envoyés
- Rendez-vous pros
- Contacts LinkedIn ajoutés
- Candidatures envoyées
- Prospects contactés
- Heures client / création
- Heures admin
- Posts publiés
- Portfolio amélioré
- Opportunités à venir

## Usage

1. Mets à jour la journée directement depuis le `Dashboard` avec les boutons `−` et `+`.
2. Change la date du bloc `Journée` pour corriger une journée passée.
3. Utilise `Historique` puis `Modifier` pour reprendre une journée en saisie détaillée.
4. Ajuste les objectifs dans `Objectifs`.
5. Utilise `Données` pour exporter ou importer une sauvegarde JSON.

Une date correspond à une seule entrée : si tu réenregistres la même journée, elle est mise à jour.

## Sauvegarde

Le stockage est local au navigateur. Pour sécuriser tes données, exporte régulièrement un fichier JSON depuis l'onglet `Données`.
